# Daniel Semindu - Software Developer Portfolio

Welcome to my personal portfolio website! This site showcases my skills, projects, and contact information as a software developer based in Dar es Salaam, Tanzania.

## Features

- **Responsive Design:** Works seamlessly on desktop and mobile devices.
- **Smooth Navigation:** Fixed navbar with smooth scrolling between sections.
- **Animated Hero Section:** Eye-catching introduction with a typing effect and parallax background.
- **About Me:** Learn about my background, tech stack, and passion for building solutions for Tanzania.
- **Skills & Expertise:** Highlights my core competencies in mobile, web, and Python development.
- **Featured Projects:** Showcases real-world projects, including Android apps and web platforms.
- **Contact Form:** Easily get in touch for collaborations or opportunities.
- **Social Links:** Connect with me on GitHub, LinkedIn, Twitter, and via email.

## Technologies Used

- **HTML5 & CSS3:** Modern, semantic markup and custom styles.
- **JavaScript:** Interactive effects, animations, and form handling.
- **Responsive Grid & Flexbox:** For layouts and mobile compatibility.
